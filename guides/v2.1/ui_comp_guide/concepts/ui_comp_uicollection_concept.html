What is uiCollection

The uiCollection library class should be used as a base class by any components that contain a collection of child UI components. uiCollection inherits from the uiElement class.

uiCollection source code is <UI_Module_dir>/view/base/web/js/lib/core/collection.js, in the Magento CE github repository: app/code/Magento/Ui/view/base/web/js/lib/core/collection.js.

uiCollection template

The uiCollection template is <UI_Module_dir>/view/base/web/templates/collection.html, in the Magento CE github repository: app/code/Magento/Ui/view/base/web/templates/collection.html.
