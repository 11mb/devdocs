Introduction

This guide helps you create and configure an Android app for Magento MobileConnect. Your app enables customers to view and make purchases for your store.

We provide you with a sample app in TBD that you can open as an Eclipse project to do the customization.

Prerequisites

Before you begin:

  1. You must be familiar with Android app development.
    To come up to speed, you can check out the Android training resources.
  2. You must know the App Code for your Magento MobileConnect application.
    To view the code, log in to the Admin Panel as an administrator and click MobileConnect > Manage Apps. You must know the value displayed in the App Code column.

Creating the Android App

To create the Android app:

  1. Open the sample Android app in Eclipse.
    The sample app is located in TBD.
    Be sure to import all related library projects, such as google-play-services_lib and android-support-v7-cardview. These libraries are available from the Android SDK Manager.
  2. Locate the <Magento sample app dir>/res/values/strings.xml resource file
  3. Change the value of base_url to your store's base URL (for example, http://mystore.example.com).
  4. Change the value of app_code to the application code displayed in the Magento Admin Panel.
    To view the code, log in to the Admin Panel as an administrator and click MobileConnect > Manage Apps. Enter the value displayed in the App Code column.
  5. Build and run your app.

Customizing Your Android App

The following sections discuss how you can customize your Magento MobileConnect Android app.

Customizing the Launch Images

The application loading screen contains 2 images: background and foreground.

These images are located in <Magento sample app dir>/res/drawable.

Customizing the App Icon and Other Images

App icons and other graphics resources are located in <Magento sample app dir>/res/drawable-* set of folders.

Every folder contains images specific to a particular screen resolution and pixel density.

Customizing Other Resources

  • Colors are defined in the <Magento sample app dir>/res/values/colors.xml file.
  • The application name is defined in the <Magento sample app dir>/res/values/strings.xml file.
